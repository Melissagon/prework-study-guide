Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • h1 is the largest of the header texts and should only be used in one place on the page.
  • ul indicates an unordered list.
  • ol indicates an ordered list.
  • li indicates a list item.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Color codes for html can be found on: https://htmlcolorcodes.com/
  • An astrisk before a code applies the rule to the whole document.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates new branch and switches to it
  • code . opens VS code to the branch you are currently in
  • git branch will show you all branches
  • git chekout and name of branch will take out out of current branch and into the branch you coded.
  • to merge all aspects of one branch into another, go into the branch you want added to then code: git merge name of other branch that conatins the info
